# Background

`-` Define um fundo para o nosso elemento

`-` Sua área de atuação é a caixa toda

`-` por padrão, é tranparente

## Exemplos

`-` Usa cores sólidas

`-` Usa imgagens

`-` Controla

- a posição das imagens
- se elas se repetem ou não
- o tamanho delas na caixa

`-` Usa cor e imagem juntas
`-` Usa cor gradiente
