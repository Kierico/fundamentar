# SQL
## 07 - Operadores Matemáticos

```sql
    -- +, adição.
    SELECT * FROM aluno WHERE matricula = 1 + 1

    -- -, subtração.
    SELECT * FROM aluno WHERE matricula = 2 - 1

    -- *, multiplicação.
    SELECT * FROM aluno WHERE matricula = 2 * 1

    -- /, divisão.
    SELECT * FROM aluno WHERE matricula = 3 / 1

    -- %, módulo ou resto da divisão.
    SELECT * FROM aluno WHERE matricula = 5 % 2
```
