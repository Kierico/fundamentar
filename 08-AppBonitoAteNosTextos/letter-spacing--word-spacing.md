# Letter-spacing e Word-spacing

## letter-spacing

* Define o espaçamento entre os caracteres.

<br>

~~~CSS
    /* CSS */
    p {
        letter-spacing: 4px;    /* <---- */
    }
~~~

## word-spacing

* Define o espaçamento entre palavras.

<br>

~~~CSS
    /* CSS */
    p {
        word-spacing: 1em;    /* <---- */
    }
~~~
