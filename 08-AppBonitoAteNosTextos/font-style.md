# Font Style

* Estilo da fonte

Value:	`normal` | `italic` | `oblique`

<br>

~~~CSS
    /* CSS */
    span {
        font-family: "Time New Roman", sans-serif;
        font-weight: normal;
        font-style: italic;    /* <--- */
    }
~~~
