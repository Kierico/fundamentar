# Font Size

* O tamanho da fonte

Value:	`<absolute-size>` | `<relative-size>` | `<length-percentage>`

<br>

~~~CSS
    /* CSS */
    p {
        font-family: "Time New Roman", sans-serif;
        font-weight: normal;
        font-style: italic;
        font-size: 30px;    /* <--- */
    }
~~~
