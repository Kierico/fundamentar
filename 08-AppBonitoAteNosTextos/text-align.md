# Text-align

* Alinhamento de um texto
* Valores: `start` | `end` | `left` | `right` | `center` | `justify` | `match-parent`

<br>

~~~CSS
    /* CSS */
    p {
        text-align: center;    /* <--- */
    }
~~~
