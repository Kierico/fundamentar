# Text-shadow

* Sombra aplicada a um texto.
* Permite multiplos valores.

<br>

~~~CSS
    /* CSS */
    p {              /* offset-x | offset-y | blur-radius | color */
        text-shadow: 1px 1px 1px red, 
                     2px 2px 1px green;    /* <--- */
    }
~~~
